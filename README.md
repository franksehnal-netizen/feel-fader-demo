# Feel Fader — Web Configurator (test build)

Live: https://franksehnal-netizen.github.io/feel-fader-demo/

Browser configurator for the Feel Fader USB MIDI controller (Acoustic Empire).

**Requires Chrome or Edge** (uses Web MIDI + Web Serial; not supported in Firefox/Safari).

This is a test/demo deployment of the single-file app for hardware testing.
